🚀 DEPLOY GUIDE (EC2 + 1 DOMAIN ONLY + VNPAY SANDBOX)
🎯 Kiến trúc đơn giản bạn sẽ dùng
Internet
   |
   |  https://yourdomain.com
   |
[ EC2 + Nginx Reverse Proxy ]
   |----------------------|
   |                      |
Frontend (React build)   Backend API (Spring Boot)
served by Nginx          localhost:5000

👉 Chỉ cần 1 domain duy nhất:

https://yourdomain.com
A. Chuẩn bị
1) EC2 instance
Ubuntu 22.04 (khuyên dùng)
Open ports:
22 (SSH)
80 (HTTP)
443 (HTTPS)
2) Domain

Ví dụ:

yourdomain.com
3) VNPAY Sandbox cần
TMN_CODE
HASH_SECRET
PAY_URL sandbox
B. DNS setup (QUAN TRỌNG)

Trỏ domain về EC2:

Nếu dùng Route53 hoặc DNS provider khác:
Type: A record
Name: yourdomain.com
Value: EC2_PUBLIC_IP

👉 Không cần subdomain gì cả.

C. Cài môi trường trên EC2

SSH vào EC2:

sudo apt update
sudo apt install nginx docker.io docker-compose -y
sudo systemctl enable nginx
sudo systemctl start nginx
D. Deploy backend (Spring Boot)
1) Run backend container

Ví dụ backend chạy port 5000:

docker run -d \
  --name hotel-backend \
  -p 5000:5000 \
  -e SPRING_PROFILES_ACTIVE=prod \
  -e VNPAY_TMN_CODE=xxx \
  -e VNPAY_HASH_SECRET=xxx \
  -e VNPAY_PAY_URL=https://sandbox.vnpayment.vn/paymentv2/vpcpay.html \
  your-backend-image
E. Build frontend (React/Vue)
1) Build local hoặc CI
npm run build
2) Copy build lên EC2
scp -r dist/ ubuntu@EC2:/home/ubuntu/frontend
3) Move vào nginx folder
sudo mkdir -p /var/www/frontend
sudo cp -r frontend/* /var/www/frontend/
F. Config Nginx (QUAN TRỌNG NHẤT)

Tạo file:

sudo nano /etc/nginx/sites-available/app
🔥 Config chuẩn:
server {
    listen 80;
    server_name yourdomain.com;

    root /var/www/frontend;
    index index.html;

    # FRONTEND (SPA)
    location / {
        try_files $uri /index.html;
    }

    # BACKEND API
    location /api/ {
        proxy_pass http://localhost:5000/;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}

Enable config:

sudo ln -s /etc/nginx/sites-available/app /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
G. HTTPS (khuyên dùng)

Cài certbot:

sudo apt install certbot python3-certbot-nginx -y

Generate SSL:

sudo certbot --nginx -d yourdomain.com
H. VNPAY CONFIG (CỰC QUAN TRỌNG)
1) Return URL (frontend)
https://yourdomain.com/customer/booking/result
2) IPN URL (backend)
https://yourdomain.com/api/public/payments/vnpay-ipn

👉 Vì backend đang chạy qua Nginx path /api

I. ENV frontend
VITE_API_BASE_URL=https://yourdomain.com/api
J. ENV backend
SERVER_PORT=5000

VNPAY_TMN_CODE=xxx
VNPAY_HASH_SECRET=xxx
VNPAY_PAY_URL=https://sandbox.vnpayment.vn/paymentv2/vpcpay.html

CORS_ALLOWED_ORIGINS=https://yourdomain.com
K. Flow hoạt động thực tế
1. User đặt phòng
https://yourdomain.com
2. Frontend gọi API
https://yourdomain.com/api/bookings
3. Backend tạo payment URL VNPAY
4. Redirect sang VNPAY sandbox
5. Thanh toán xong:
VNPAY -> redirect:
https://yourdomain.com/customer/booking/result
6. VNPAY gửi IPN:
POST https://yourdomain.com/api/public/payments/vnpay-ipn
